### Layer 1

* [Glamsterdam](https://forkcast.org/upgrade/glamsterdam/) upgrade (targeting June 2026):  
  * Scoping continues January 5 at all core devs \- execution (ACDE) [\#227](https://github.com/ethereum/pm/issues/1854)  
    * [16 EIPs Proposed for Inclusion](https://forkcast.org/upgrade/glamsterdam/#proposed-for-inclusion) yet to be decided upon  
    * [14 EIPs Considered for Inclusion](https://forkcast.org/upgrade/glamsterdam/#considered-for-inclusion)  
* [Hegotá](https://forkcast.org/upgrade/hegota/) upgrade:  
  * Fork focus & headliner discussions start January 8 at all core devs \- consensus (ACDC) [\#172](https://github.com/ethereum/pm/issues/1844)  
* Ethereum improvement proposals (EIPs):  
  * Forkcast [EIP directory](https://forkcast.org/eips/): view/filter list, details include governance timeline  
  * [EIP8115](https://github.com/ethereum/EIPs/pull/10997/files): Batch priority fees at end of block (split from [EIP7799](https://eips.ethereum.org/EIPS/eip-7799): System logs)  
  * [EIP8116](https://github.com/ethereum/EIPs/pull/10998/files): Replace cumulative receipt fields (split from [EIP6466](https://eips.ethereum.org/EIPS/eip-6466): SSZ receipts)
